# Project Template

This template outlines the recommended structure for an individual AI filmmaking project within this repository. Each project should have its own dedicated folder following this structure.

## Project Structure:

\`\`\`
[Your-Film-Title]/
├── README.md                 # Project-specific overview and synopsis
├── Pre-Production/           # Contains character sheets, storyboards, mood boards, script/shot list
│   ├── characters/           # Copies or links to character sheets used in this project
│   ├── locations/            # Descriptions and prompts for specific locations
│   ├── props/                # Descriptions and prompts for specific props
│   ├── storyboards/          # Visual storyboards or shot breakdowns
│   └── script.md             # The film's script or detailed shot list
├── Generation/               # Contains prompts used, generation logs, and raw generated assets
│   ├── prompts/              # Specific prompts used for this project, categorized
│   ├── raw_assets/           # Unedited generated video clips and images
│   └── generation_log.md     # Notes on generation iterations, successful prompts, challenges
└── Post-Production/          # Contains edited sequences, sound design, VFX, and final exports
    ├── edits/                # Project files for video editing software
    ├── sound_design/         # Audio assets, music, sound effects
    ├── vfx/                  # Visual effects assets or project files
    └── final_export/         # Final rendered film files
\`\`\`
